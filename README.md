# Twitch Bot

A simple bot for twitch with [tmi.js](https://github.com/tmijs/tmi.js)

## .env

| .env file    | Value                                |
| ------------ | ------------------------------------ |
| BOT_USERNAME | bot name                             |
| OAUTH_TOKEN  | [oauth](https://twitchapps.com/tmi/) |
| CHANNEL_NAME | channel name                         |
